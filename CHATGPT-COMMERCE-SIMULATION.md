## 📊 Backend Workflow (Behind the Scenes)

### When Customer Clicks "Buy Now" in ChatGPT:

```
1. ChatGPT → Stripe Checkout
   ├─ Customer pays $39.92
   └─ Stripe creates charge

2. Stripe → Webhook to GangRun
   ├─ POST /webhooks/acp/order_created
   ├─ Order data: {
   │    "offer_id": "chicago-postcards-4x6-glossy-500",
   │    "customer_email": "iradwatkins@gmail.com",
   │    "amount": 39.92,
   │    "currency": "USD"
   │  }
   └─ GangRun creates order in database

3. GangRun → Email to Customer
   ├─ Send upload link
   └─ Order confirmation

4. Customer → Uploads Design
   ├─ Files stored in MinIO
   └─ Status: PENDING_PROOF

5. GangRun → Vendor
   ├─ Send files + specs
   ├─ Vendor creates proof
   └─ Proof emailed to customer

6. Customer → Approves Proof
   ├─ Status: IN_PRODUCTION
   └─ Vendor begins printing

7. Vendor → Ships Order
   ├─ Status: SHIPPED
   ├─ Tracking sent to customer
   └─ Payment sent to vendor

8. Customer → Receives Order
   ├─ Status: DELIVERED
   └─ Feedback request sent
```
